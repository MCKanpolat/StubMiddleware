StubMiddleware is an open source stub data generator for .Net Core & Asp.Net Core.
 
## Installation

    PM> Install-Package StubMiddleware

### Dependencies

    - Faker.Net

### Building the project

    dotnet build

### Run tests 
 

    dotnet test
